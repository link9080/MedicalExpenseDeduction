# 初期設定
```
# ログイン
clasp login

# 既にあるプロジェクトからクローン
clasp clone {プロジェクトID}


```


# clasp使いかた


GASへアップロード: 
```
clasp push
```
GASからダウンロード: 
```
clasp pull
```

変更を監視して自動アップロード: 
```
clasp push --watch
```

GASエディタで開く
```
clasp open
```
デプロイ済みURL取得
```
clasp deployments
```

デプロイ
```
clasp deploy
```
